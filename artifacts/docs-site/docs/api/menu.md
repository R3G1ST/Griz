# Menu API

## GET /api/v1/menu

Получить все позиции меню.

```bash
curl https://api.grizzly-lounge.qmbox.ru/api/v1/menu
```

## GET /api/v1/menu/:id

Получить позицию по ID.

## POST /api/v1/menu

Создать новую позицию (требуется API ключ).

## PUT /api/v1/menu/:id

Обновить позицию (требуется API ключ).

## DELETE /api/v1/menu/:id

Удалить позицию (требуется API ключ).
