# WebSocket

API поддерживает WebSocket для получения мгновенных уведомлений в реальном времени.

## Подключение

Подключайтесь к wss://api.grizzly-lounge.qmbox.ru/ws с JWT токеном в query-параметре.

Формат: wss://api.grizzly-lounge.qmbox.ru/ws?token=ВАШ_JWT_ТОКЕН

## События

| Событие | Описание |
|---------|----------|
| connected | Успешное подключение |
| new_post | Новый пост опубликован |
| menu_updated | Меню обновлено |

## Пример для Telegram-бота (Node.js)

Используйте библиотеку ws для подключения. При получении события new_post отправляйте уведомление в чат.

## Пример для Python (aiogram)

Используйте библиотеку websockets для асинхронного подключения. Парсите JSON-сообщения и реагируйте на тип события.
